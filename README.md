# Hash Calculator
> Lite tool to calculate hashsum of any file

![Preview](https://i.imgur.com/akkJGYq.png)

## Supported hash-algorithms
* MD5
* SHA1
* SHA224
* SHA256
* SHA384
* SHA512

## Install dependencies
```shell
pip3 install -r requirements.txt
```

## Start application
```shell
python main.py
```
